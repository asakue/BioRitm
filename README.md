## Features

- **Shopping Cart:** A fully functional shopping cart to purchase services.
- **Persistent Cart:** Cart data is saved, so you don't lose your items between sessions.
- **AI-Powered Recommendations:** Get intelligent service recommendations based on the items in your cart.
- **Modern Design:** A clean, responsive, and visually appealing design built with Next.js and Tailwind CSS.

To get started, run the development server:

```bash
npm run dev
```
